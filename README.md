# Projeto_Avaliacao
https://www.figma.com/make/LjN6DERS8j6VNBNAsRce4Q/Tabela-da-Copa-do-Mundo?t=eAYfQaDSezs4BqwM-20&fullscreen=1

Este site mostra uma versão fictícia da final da Copa do Mundo de 2026, onde o Brasil vence a Alemanha por 7 a 1 e conquista o tão sonhado hexacampeonato. A página foi criada para simular uma cobertura esportiva completa, trazendo informações sobre a partida, estatísticas, destaques dos jogadores e os principais acontecimentos do jogo. O objetivo é fazer com que o visitante tenha a sensação de estar acompanhando uma notícia real, podendo conferir todos os detalhes desse resultado histórico e imaginário. Além do placar final, o site também apresenta dados da partida, informações sobre os autores dos gols, assistências, desempenho das equipes e a trajetória até a conquista do título.

O botão " Gols da Partida" mostra todos os gols marcados durante o jogo, informando o minuto em que aconteceram, quem marcou e quem deu a assistência. Com isso, o usuário consegue acompanhar como o placar foi sendo construído ao longo da partida.

Já o botão " Chaveamento Completo" permite visualizar todo o caminho das seleções na Copa do Mundo, mostrando os confrontos das fases eliminatórias, os resultados de cada jogo e como os times chegaram até a grande final.
